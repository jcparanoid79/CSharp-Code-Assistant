# Gemini AI Coding Assistant

This is a simple web application built with Flask that utilizes the Google Gemini API to provide coding assistance.

## Features

- Generate Code
- Debug Code
- Explain Code
- Suggest Refactoring
- Write Documentation
- Dark/Light Mode Toggle

## Setup

1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Create a `.env` file in the project root with your Gemini API key: `GEMINI_API_KEY=YOUR_API_KEY`
4. Run the application: `python gemini_agent_app.py`

## Usage

Open your browser to `http://127.0.0.1:5000` and use the interface to get coding assistance.
